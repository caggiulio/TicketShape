# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),

## [1.0.0] - 2025-11-14

### Added
- Initial release of TicketShape
- `TicketShape`: Custom shape with semi-circular cutouts
- `TicketDashedLine`: Horizontal dashed separator line
- `TicketShapeView`: Complete ticket container view with content builder
- Fully customizable parameters for all components
- Convenience modifiers for easier customization
- Comprehensive documentation and examples
- Swift Package Manager support
- Preview examples for multiple use cases

### Features
- Customizable cutout position (0.0-1.0)
- Adjustable cutout radius
- Configurable corner radius
- Custom fill colors
- Customizable dashed line appearance
- Optional dashed line visibility
- Responsive design that adapts to any frame size
- Built with SwiftUI for iOS 15.0+, macOS 12.0+, tvOS 15.0+, watchOS 8.0+
